Detailed Description

The structure node contains requried data for a node needed to implement the algorithm.

Parameters

rlinkContains address of the right child of the node.
xStores the x coordinate of the node
yStores the y coordinate of the node
offsetSpecifies the horizontal offset between the node and each of its sons
valValue of the key held by the node.
threadServes as a flag to distinguish between a regular and a temporary pointer
